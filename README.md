Nightingale C++ — це унікальний заголовочний файл (DSL), який дозволяє писати повноцінний, робочий код на C++ українською мовою. Цей експериментальний проект перетворює строгий технічний синтаксис на природний та читабельний текст, повністю зберігаючи при цьому швидкодію та потужність оригінального компілятора.



Особливості проекту

Бібліотека охоплює всі базові аспекти мови. Вона містить переклади для ключових слів, типів даних, керуючих конструкцій та логічних операторів. Для зручності написання алгоритмів реалізовано повний набір числівників від нуля до мільярда з урахуванням родових закінчень. Особливістю архітектури є розумна інтеграція зі стандартною бібліотекою C++: макроси для роботи з часом, багатопоточністю, файловою системою та математичними функціями активуються автоматично лише тоді, коли ви підключаєте відповідні заголовки у своєму проекті.



Встановлення та використання

Для початку роботи достатньо завантажити файл NightingaleСXX.hpp та покласти його в папку вашого проекту. Вкрай важливо підключати цей файл суворо після всіх інших стандартних бібліотек (наприклад, після <iostream> чи <vector>). Це архітектурне правило гарантує, що україномовні макроси не зламають внутрішній код стандартних інструментів компілятора.
